# sudoku_solver
A Silly Sudoku Solver

To run the sudoker solver, do:

```bash
perl sudoku.pl <GAME>
```

Where '''<GAME>''' is a game package like: '''game::easy1''', or '''game::bad'''.

To create a sudoku board, see the '''game''' folder for examples on how to do it.
